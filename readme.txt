== 説明 ==
VK Bogo Language Switcher は Bogo の言語切替リストにカスタムクラスやスタイル設定を追加する拡張プラグインです。

* 表示スタイルの切替（国旗とテキスト／テキストのみ）
* 並び方向（横並び／縦積み）
* 表示中の言語を非表示
* 上記設定に応じたクラスを `<ul class="bogo-language-switcher">` に付与
* ブロックエディター（サイトエディター含む）でも同じクラスと CSS を適用

== インストール ==
1. プラグインを `wp-content/plugins/vk-bogo-language-switcher` にアップロードします。
2. 管理画面の「プラグイン」で VK Bogo Language Switcher を有効化します。  
   ※ Bogo が有効化されている必要があります。
3. 管理画面「Bogo > スタイル」で表示スタイルを設定してください。
